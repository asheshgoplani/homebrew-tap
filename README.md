# Homebrew Tap for Agent Deck

This is the official Homebrew tap for [Agent Deck](https://github.com/asheshgoplani/agent-deck), a terminal session manager for AI coding agents.

## Installation

```bash
brew install asheshgoplani/tap/agent-deck
```

Or add the tap first:

```bash
brew tap asheshgoplani/tap
brew install agent-deck
```

## Updating

```bash
brew upgrade agent-deck
```
